# Node.js 下载安装指南

## 🚀 快速安装步骤

### 1. 下载 Node.js
- 浏览器应该已经打开了 Node.js 官方下载页面
- 选择 **LTS 版本**（推荐，更稳定）
- 下载适合 Windows 的 `.msi` 安装包

### 2. 安装 Node.js
- 双击下载的 `.msi` 文件
- 按照安装向导提示进行安装
- **重要**：确保勾选 "Add to PATH" 选项

### 3. 验证安装
安装完成后，重新打开命令提示符，运行以下命令验证：

```bash
node --version
npm --version
```

如果显示版本号，说明安装成功！
